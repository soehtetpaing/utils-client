Genius Colors

All color are mixture of Red, Green & Blue light!

Colors can be specified in
  • name (eg. style = "color: red";)
  • rgb(R, G, B)
    * RGB ~ 0 to 225 (eg. style = "color: rgb(225, 0, 0)";)
  • hexadecimal #RRGGBB #RGB
    * RGB ~ 0 to F/f (eg. style = "color: #ff0000/#f00";)
  • keyword currentcolor 
    (e.g
    #myDiv {
        color: red;
        border: 10px solid currentcolor;
    }
)
* hue ~ 0 to 360 (color wheel) in hsl in CSS3/hwb in CSS4
* saturation, lightness, whiteness, blackness, darkness ~ 0% to 100%
  • hsl( hue, saturation, lightness) (eg. style = "color: hsl(0, 100%, 50%)";)
w3color.js with data-w3-color = "" in CSS4
.html
<script src = "assets/js/w3color.js" ></script>
<script> w3SetColorsByAttribute(); </script>
  • hwb(hue, whiteness, blackness) (eg. data-w3-color = "hwb(0, 0%, 0%) ";)
  • cmyk(cyan, magenta, yellow, key/black) (eg. data-w3-color = "cymk(100%, 0%, 0%, 0%)";)
  • NCol (Natural Color) - hue, whiteness, darkness (eg. data-w3-color = "R0, 0%, 0% ";)
